INSTRUCCIONES PARA EL ICONO:

Para generar el icono de la aplicación (icon.ico), necesita:

1. Crear o conseguir una imagen PNG de 256x256 píxeles con el logo de TrakCare

2. Convertir el PNG a ICO usando una de estas opciones:

   Opción A - Online:
   - Visitar: https://convertio.co/es/png-ico/
   - Subir su imagen PNG
   - Descargar el archivo .ico generado

   Opción B - ImageMagick (si está instalado):
   - convert logo.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico

   Opción C - GIMP (software gratuito):
   - Abrir la imagen en GIMP
   - File > Export As...
   - Guardar como "icon.ico"
   - Seleccionar todos los tamaños sugeridos

3. Guardar el archivo como: frontend/electron/icon.ico

Si no proporciona un icono personalizado, Electron usará el icono por defecto.

Para una aplicación profesional, se recomienda usar iconos de alta calidad.
