Generador de Icono TrakCare Offline

Este generador crea un icono básico de 256x256 píxeles para la aplicación.

Instrucciones:

  1. Haga click en "Descargar Icono (PNG)" para obtener el archivo PNG
  2. Convierta el PNG a ICO usando:
    • Convertio Online
    • O ImageMagick: convert icon.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico
  3. Guarde el archivo ICO como: frontend/electron/icon.ico
  4. Recompile la aplicación Electron

Nota: Para una aplicación profesional, reemplace este icono con el logo oficial de TrakCare.
